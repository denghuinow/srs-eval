# 更新日志

## [1.0.0] - 2025-11-21

### 🎉 里程碑版本

这是 srs-eval 的第一个正式里程碑版本，标志着系统已经稳定可用。

### ✨ 主要功能

- **要点提取**：从基准文档自动提取结构化要点清单
- **量化评估**：基于检查项的量化评估方法，确保结果客观、可重复
- **多维度评分**：支持完整性、准确性两个维度的评估
- **可重复性保证**：
  - 要点清单缓存机制
  - 多次提取取最优机制
  - 固定温度参数（temperature=0）
  - 多次运行取平均
- **并行执行**：支持批量评估、多次提取、多次运行的并行执行，大幅提高效率
- **多格式输出**：支持 JSON、CSV、Markdown 三种输出格式
- **批量处理**：支持批量评估多个文档
- **提示词版本管理**：支持多版本提示词管理

### 🔧 技术特性

- 基于 Python 3.10+
- 使用 uv 进行依赖管理
- 支持 OpenAI API 及兼容接口
- 完整的日志记录系统
- 自动缓存机制

### 📝 使用方式

```bash
# 安装依赖
uv sync

# 评估单个文档
uv run main.py --baseline baseline.md --target target1.md

# 批量评估
uv run main.py --baseline baseline.md --targets target1.md target2.md
```
